Szükséges előzmény: JSON

A JSON5 a JSON egy kiterjesztése. Célja a könnyebb kézzel szerkeszthetőség biztosítása úgy hogy a könnyű feldolgozhatóság is megmaradjon.

Ennek érdekében a JSON funkcióit a JavaScript lehetőségeivel egészíti ki.

Így a JSON5 továbbra is megfelelő adattárolás mellett adatkommunikációra is.
